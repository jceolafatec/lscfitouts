# Portfolio Manager PRO – Complete Setup Guide

## Overview

The enhanced portfolio manager system gives you full control over your project portfolio with:

- **Enhanced GUI Manager** (`project_manager_pro.py`) – Add, edit, delete projects with full 3D model support
- **JSON-based storage** – Easy to backup, version control, and integrate with your website
- **Dynamic website** – Projects load automatically from `assets/data/projects.json`
- **Search & filtering** – Built-in search and tag-based filtering on the portfolio page
- **3D model viewer** – Interactive WebGL viewer for GLB/GLTF models
- **Auto file management** – Images and models automatically copied to asset folders

---

## Quick Start (1 minute)

### Step 1: Run the Manager

```bash
cd website/tools
python project_manager_pro.py
```

A window opens with:
- **Left side:** List of projects
- **Right side:** Tabbed form with fields

### Step 2: Add a Project

1. Click **"➕ New"**
2. Fill in the form:
   - **Title** (required)
   - **Meta** (e.g., "UI Design · 2025")
   - **Description**
   - **Image** – Click "Browse" to pick an image (auto-copied to `assets/img/`)
   - **3D Model** – Click "Browse" to pick a `.glb`/`.gltf` file (auto-copied to `assets/models/`)
   - **GitHub/Demo Links** – Add URLs for external links
   - **Tags** – Comma-separated keywords (clickable on website)
3. Click **"💾 Save Project"**

### Step 3: Export & View

1. Click **"📄 Export All"** to:
   - Save to `assets/data/projects.json` (website will load from this)
   - Generate `projects.xml` (backup format)
   - Generate `projects_snippet.html` (for manual copying if needed)

2. Open `index.html` in a browser – projects appear dynamically!

## Features in Detail

### 🎯 Project Fields

| Field | Purpose | Example |
|-------|---------|---------|
| **Title** ⭐ | Project name | "3D Model Showcase" |
| **Meta** | Date & category | "3D Visualization · 2025" |

| **Demo Link** | Live demo URL | "https://example.com/demo" |

### 🔍 Website Features

**Search:** Type in the search box to find projects by title, description, or tags.

**Tag Filtering:** Click any tag on a project card to filter by that tag.

**3D Viewer:** If a project has a model, a "View 3D Model" button appears → opens interactive viewer.

**Responsive:** Works on desktop, tablet, mobile.

## Data Structure

### Location: `assets/data/projects.json`

```json
[
  {
    "id": "1",
    "title": "Project Name",
    "meta": "Category · Year",
    "description": "...",
    "image": "assets/img/project-01.jpg",
    "model_path": "assets/models/model-01.glb",
    "github_link": "https://...",
    "demo_link": "https://...",
    "tags": ["3D", "WebGL"]
  }
]
```

This file is auto-generated by the manager. **Do not edit manually** unless you know JSON syntax.

## Common Tasks

### ✅ Edit an Existing Project

1. Open manager: `python project_manager_pro.py`
2. Click project name in the list
3. Change fields
4. Click **"💾 Save Project"**
5. Click **"📄 Export All"** to update the website
6. Refresh your browser

### ✅ Delete a Project

1. Click project name
2. Click **"❌ Delete"**
3. Confirm deletion
4. Click **"📄 Export All"**

### ✅ Replace a Project Image

1. Click project
2. In **"Media & Files"** tab, click browse next to Image
3. Select new image (auto-copies)
4. Click **"💾 Save Project"** and **"📄 Export All"**

### ✅ Update All Projects at Once

1. Make changes to each project
2. Click **"📄 Export All"** once when done
3. Refresh website

### ✅ Backup Your Projects

```bash
# Copy the JSON file
cp assets/data/projects.json projects-backup-$(date +%Y%m%d).json
```

The JSON file is human-readable and version-control friendly (Git).

## Troubleshooting

### "File not found" error

**Solution:** Check that the image/model file path exists and is readable.

### Projects don't appear on website

**Solution:**
1. Make sure you ran **"📄 Export All"** in the manager
2. Check that `assets/data/projects.json` exists
3. Open browser console (F12) and look for error messages
4. Hard-refresh the page (Ctrl+Shift+R or Cmd+Shift+R)

### Images/models not showing

**Solution:**
1. Make sure files were copied to `assets/img/` and `assets/models/`
2. Check file permissions (especially on macOS/Linux)
3. Use "Browse" button in manager to ensure proper paths

### Want to use the old CLI tool?

Run: `python project_manager.py` (but manager_pro is recommended)

## Advanced: Convert Old XML to JSON

If you have an old `projects.xml`:

```bash
cd website/tools
python migrate_to_json.py
```

This reads `projects.xml` and writes to `assets/data/projects.json`.

## Packaging as an Executable

To create a standalone `.exe` (Windows) or `.app` (macOS):

```bash
# Install PyInstaller
pip install pyinstaller

# Create executable
pyinstaller --onefile project_manager_pro.py

# Result: dist/project_manager_pro.exe (or .app on macOS)
```

No Python installation needed after this! Share the `.exe` with others.

## Tips & Best Practices

- **Keep descriptions short** (2-3 sentences) for best layout
- **Use consistent tags** (e.g., "3D" not "3d" and "3-D")
- **Test image sizes** – crop to 16:9 or square for best appearance
- **Always click "Export All"** after making changes
- **Backup JSON regularly** (commit to Git if using version control)
- **Browser cache:** Hard-refresh (Ctrl+Shift+R) if changes don't appear
